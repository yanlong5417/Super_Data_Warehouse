# 超级个体知识库 — 结构概览

这是一个基于 **PARA 方法** 构建的个人知识库。

## 目录说明

| 目录            | 用途                      |
| ------------- | ----------------------- |
| `000-收件箱`     | 临时想法、剪藏、未整理的碎片笔记        |
| `100-领域`      | 持续关注的领域（投资、编程、写作、健康...） |
| `200-项目`      | 有明确目标的短期项目              |
| `300-资源`      | 书籍、课程、工具、参考文档           |
| `400-日记`      | 每日记录、周复盘                |
| `500-输出`      | 写作成品、分享稿、产品想法           |
| `600-元知识`     | 关于知识库本身的笔记（使用指南、模版）     |
| `900-存档`      | 已完成或不再活跃的笔记归档           |
| `attachments` | Obsidian 附件（图片、PDF 等）   |

## 核心工作流

1. **捕获** → 随手记入 `000-收件箱`
2. **归类** → 定期整理到各领域/项目/资源
3. **连接** → 使用 `[[双向链接]]` 关联相关笔记
4. **提炼** → 把信息转化为个人知识
5. **输出** → 在 `500-输出` 中完成创作
