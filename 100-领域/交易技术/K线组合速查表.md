# K 线组合速查表

创建时间: 2026-07-12

## 单根反转信号

| 名称 | 形态特征 | 含义 |
|------|---------|------|
| 锤子线 | 下影线>=实体2倍，上影线短或无 | 底部反转，跌势可能结束 |
| 上吊线 | 同上，但出现在上涨后 | 顶部反转，涨势可能结束 |
| 十字星 | 开盘=收盘，上下影线 | 多空平衡，趋势可能反转 |
| 长腿十字 | 上下影线都很长 | 剧烈分歧，变盘信号 |
| 墓碑线 | 上影线长，无下影线 | 顶部反转，空头力量强 |
| 纺锤线 | 实体小，影线适中 | 犹豫信号，趋势可能减弱 |

## 双根组合

| 名称 | 形态 | 含义 |
|------|------|------|
| 看涨吞没 | 阴线后阳线，阳线实体包裹阴线 | 强反转信号 |
| 看跌吞没 | 阳线后阴线，阴线实体包裹阳线 | 强顶部信号 |
| 刺透形态 | 阴线后阳线，阳线突破阴线中点 | 底部反转 |
| 乌云盖顶 | 阳线后阴线，阴线跌破阳线中点 | 顶部反转 |
| 孕线 | 大实体后小实体，小实体被包裹 | 趋势减弱，可能反转 |

## 三根及以上组合

| 名称 | 形态 | 含义 |
|------|------|------|
| 晨星 | 长阴+小实体+长阳 | 底部反转 |
| 黄昏星 | 长阳+小实体+长阴 | 顶部反转 |
| 三只乌鸦 | 三连阴，逐步走低 | 顶部确认 |
| 红三兵 | 三连阳，逐步走高 | 底部确认 |
| 上升三法 | 长阳+三小阴+长阳 | 中继上涨 |
| 下降三法 | 长阴+三小阳+长阴 | 中继下跌 |

## 结合 MACD / 均线的用法

K线信号 -> 确认反转可能
MACD背离 -> 确认方向
均线排列 -> 确认趋势
成交量放大 -> 确认信号有效

> 三条件同时满足时信号可靠性最高。
