## 学习路线（共 5 阶段，约 3 个月）

### 第一阶段：市场基础（第 1-2 周）

**目标：** 搞清你交易的是什么

- [ ] **A股市场结构**
  - 上交所/深交所/北交所 分工
  - 主板/创业板/科创板/北交所 区别与门槛
  - 交易规则：T+1、涨跌幅限制（10%/20%/30%）、集合竞价
  - 手续费结构：印花税、佣金、过户费
- [ ] **基础术语**
  - K 线（阳线/阴线/十字星/锤子线）
  - 成交量（放量/缩量/天量/地量）
  - 市值、PE（市盈率）、PB（市净率）、ROE
  - 流通盘、换手率
- [ ] **行情软件操作**
  - 通达信/东方财富/同花顺 基本操作
  - 自选股管理、板块分类、F10 看资料

---

### 第二阶段：技术分析进阶（第 3-4 周）

**目标：** 补全你已有技术分析的盲区

- [ ] **K 线组合形态**
  - 单根反转：锤子线、上吊线、吞没形态
  - 持续形态：三法、跳空缺口
- [ ] **均线系统**
  - 均线金叉死叉 vs MACD 的区别
  - 多头排列/空头排列
  - 格兰维尔八大买卖法则
  - 144均线（Vegas隧道核心）的深层原理
- [ ] **MACD 深化**
  - 已掌握：金叉死叉、顶底背离、五杀战法
  - 补充：MACD 零轴意义、MACD 与成交量配合
- [ ] **补充指标**
  - RSI（超买超卖、背离）
  - 布林带（带宽收缩放大的含义）
  - KDJ（与 MACD 配合使用）
- [ ] **量价关系**
  - 量价配合/量价背离
  - 放量突破/缩量回调/天量见顶
  - 主力资金流入流出判断
- [ ] **谐波交易**（你的 PDF 有详细资料）
  - Fibonacci 回调与扩展
  - Gartley/蝙蝠/蝴蝶/螃蟹 形态
  - 谐波模式 + MACD 共振信号

---

### 第三阶段：基本面分析（第 5-6 周）

**目标：** 从只看图到看懂公司

- [ ] **财务报表三张表**
  - 利润表：营收、毛利率、净利率、扣非净利润
  - 资产负债表：资产负债率、流动比率、商誉
  - 现金流量表：经营现金流（最重要）、投资/筹资现金流
- [ ] **估值方法**
  - PE 估值（静态/动态/TTM）
  - PB 估值（适用于银行/周期股）
  - PEG（成长股估值）
  - DCF（现金流折现，进阶）
- [ ] **行业分析框架**
  - 行业生命周期（导入-成长-成熟-衰退）
  - 产业链分析（上游/中游/下游）
  - 竞争格局（集中度/壁垒）
- [ ] **消息面解读**
  - 政策类（降准降息、产业政策）
  - 事件驱动（业绩预告、分红、减持）
  - 龙虎榜解读

---

### 第四阶段：交易系统（第 7-8 周）

**目标：** 把知识整合成可执行的系统

- [ ] **交易策略设计**
  - 确定你的交易风格（短线/波段/中长线）
  - 入场规则
  - 出场规则（止盈/止损）
  - 仓位管理
- [ ] **资金管理**
  - 凯利公式
  - 固定比例/固定金额/波动率调整
  - 最大回撤控制
- [ ] **风险控制**
  - 单笔止损限额（建议 1-2%）
  - 账户总风险限额（建议 10-15%）
  - 相关性风险
  - 黑天鹅防护
- [ ] **交易心理**
  - FOMO 与恐惧止损
  - 交易纪律：按规则执行
  - 连续亏损时的应对方案

---

### 第五阶段：实战与复盘（第 9 周起，持续）

**目标：** 用实盘验证系统

- [ ] **模拟盘转小资金实盘**
- [ ] **每日复盘**
- [ ] **每周统计（胜率/盈亏比/最大回撤）**
- [ ] **策略迭代优化**
